Hide Incompatible Applications=Nekompatibilní aplikace skrýt
This setting will hide applications that are listed as being incompatible with your version of Unraid=Tato volba skryje aplikace, které nejsou kompatibilní s tvojí verzí Unraidu
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Tato volba přidá na kartách a v popupech další tlačítko, které umožní přeinstalaci již nainstalované aplikace s výchozími hodnotami šablony. Tato funkce může být matoucí a používá se hlavně v případech, kdy chceš mít danou aplikaci spuštěnou vícekrát zároveň

Mouse wheel threshold=Práh kolečka myši
If you tilt the mouse wheel left or right, the browser returned value will change Set the threshold to the value returned Note that different browsers and operating systems may return different values=Pokud nakloníte kolečko myši doleva nebo doprava, hodnota vrácená prohlížečem se změní. Nastavte práh na tuto hodnotu. Upozorňujeme, že různé prohlížeče a operační systémy mohou vracet různé hodnoty.
This is a global value for all devices If it causes issues, then disable Mouse Wheel Page Changes=Toto je globální hodnota pro všechna zařízení. Pokud způsobuje problémy, vypněte změny stránek kolečkem myši.
