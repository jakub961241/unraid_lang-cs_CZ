Hide Deprecated Applications=Skrýt zastaralé aplikace
This setting hides deprecated applications from the list of available apps Deprecated apps will still however display within the installed and previous apps sections=Toto nastavení skryje zastaralé apllikace ze seznamu dostupných aplikací. Zastaralé aplikace se ale stále budou zobrazovat v sekcich nainstalované aplikace a předchozí aplikace
Hide Incompatible Applications=Skrýt nekompatibilní aplikace
This setting will hide applications that are listed as being incompatible with your version of Unraid=Toto nastavení skryje aplikace, které nejsou kompatibilní s vaší verzí Unraidu
Allow install of second instance=Povolit instalaci druhé instance
This option enables another button on the cards and popups which will allow you to reinstall an installed application using its default template values This option can be confusing at times, and is only generally used if you wish to install and have an application running twice simultaneously=Toto nastavení povolí další tlačítko na kartách a vyskakovacích oknech, které vám povolí přeinstalovat aplikaci při použití její výchozí šablony a hodnot. Toto nastavení může být někdy matoucí a je obecně používáno, pokud chcete instalovat a mít aplikaci spuštěnou vícekrát najednou
Show descriptions on application cards=Zobrazovat popisy na kartách aplikace
This option dictates whether the descriptions appear on the cards or not=Toto nastavení určuje jestli se budou zobrazovat popisy na kartách, nebo ne
Allow CA to check for updates to applications=Povolit KA kontrolu aktualizací aplikací
This setting enables CA to check for updates to applications Note that Community Applications will always check for updates to itself=Tímto nastavením povolíte KA kontrolovat aktualizace aplikací. Komunitní Aplikace vždy kontroluje aktualizace sebe sama.
Allow CA to send any emergency notifications=Povolit KA odesílat nouzová upozornění
This setting allows CA to periodically scan your system and alert you to the presence of=Tímto nastavením povolíte KA periodicky skenovat váš systém a upozorňovat vás na přítomnost
1 Known malware and security vulnerabilities within applications=1) Známého malware a bezpečnostních chyb v aplikací
2 Critical security vulnerabilities within Unraid OS=2) Kritických bezpečnostních chyb v Unraid OS
3 Any installed applications which may seriously hinder the operation of your server=3) Jakýchkoliv instalovaných aplikací, který můžou mít negativní vliv na funkci serveru
Save CA debugging information=Uložit ladící informace KA
Only enable this when advised to from the author of Community Applications=Toto povolte pouze pokud vás k tomu vyzve autor Komunitních Aplikací
Enable developer mode=Povolit vývojářský režim
This unlocks some extra features within CA generally only used by developers Not recommended to enable=Tímto odemknete několik dalších funkcí v KA. Tyto jsou většinou používány při vývoji. Nedoporučuje se zapínat.
